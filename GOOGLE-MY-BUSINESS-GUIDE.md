## OBTENIR DES AVIS GOOGLE (CRITIQUE)

### Stratégie pour obtenir des avis :

1. **Après chaque chantier satisfaisant**, envoie ce SMS :
```
Bonjour [Prénom], merci pour votre confiance ! Si vous êtes satisfait de notre travail, un petit avis Google nous aiderait beaucoup : [LIEN AVIS]. Merci et à bientôt ! - Élagage Abattage 37
```

2. **Lien direct vers les avis** (à créer après validation de la fiche) :
   - Va dans ton compte GMB > Accueil > "Obtenir plus d'avis"
   - Copie le lien court fourni par Google

3. **Objectif** :
   - Semaine 1-2 : 10 avis minimum
   - Mois 1 : 25 avis
   - Mois 3 : 50+ avis

4. **Répondre à TOUS les avis** (positifs et négatifs)

### Exemple réponse avis positif :
```
Merci beaucoup [Prénom] pour votre avis ! C'était un plaisir d'intervenir chez vous pour l'élagage de votre [type d'arbre]. N'hésitez pas à nous recontacter pour l'entretien annuel. À bientôt !
```

### Exemple réponse avis négatif :
```
Bonjour [Prénom], nous sommes désolés que notre prestation n'ait pas répondu à vos attentes. Nous prenons votre retour très au sérieux. Pourriez-vous nous contacter au 06 31 46 43 50 pour que nous puissions en discuter et trouver une solution ? Cordialement.
```

## VÉRIFICATION DE LA FICHE

Google va te demander de vérifier ton entreprise. Options :

1. **Par courrier** (5-14 jours) - Une carte postale avec code
2. **Par téléphone** - Appel automatique avec code
3. **Par email** - Si disponible pour ton type d'entreprise
4. **Instantanée** - Si ton site est déjà indexé avec les mêmes infos

**Recommandation** : Choisis la vérification par téléphone si disponible.
